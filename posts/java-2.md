---
title: "Các khái niệm cơ bản về OOP trong Java"
date: "2024-12-02"
---
OOP (Object-Oriented Programming) trong Java bao gồm bốn trụ cột: Kế thừa, Đóng gói, Đa hình, và Trừu tượng. Ví dụ, class và interface giúp triển khai trừu tượng. Java hỗ trợ kế thừa đơn giản để tránh phức tạp. Hãy thử viết một class đơn giản để hiểu rõ hơn.
