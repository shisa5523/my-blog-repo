## Python là gì?

Python là một ngôn ngữ lập trình cấp cao, dễ học và dễ sử dụng. Nó được tạo ra bởi Guido van Rossum vào năm 1989. Python nổi tiếng với cú pháp đơn giản, dễ đọc, và có thể chạy trên nhiều nền tảng khác nhau (Windows, Mac, Linux).

## Tại sao nên chọn Python cho Data Analysis?

1. **Cú pháp đơn giản**: Python có cú pháp gần giống với tiếng Anh, rất dễ học
2. **Thư viện mạnh mẽ**: Có những thư viện như Pandas, NumPy, Matplotlib cho Data Analysis
3. **Cộng đồng lớn**: Có rất nhiều tài liệu, tutorial, và cộng đồng hỗ trợ
4. **Linh hoạt**: Có thể dùng cho web, machine learning, data analysis, automation
5. **Free & Open Source**: Hoàn toàn miễn phí

## Cài đặt Python

Để cài đặt Python trên máy tính của bạn:
1. Truy cập https://www.python.org/downloads/
2. Tải phiên bản Python mới nhất (3.11 hoặc cao hơn)
3. Chạy file installer và chọn "Add Python to PATH"
4. Verify: Mở Command Prompt/Terminal, gõ `python --version`

## Các khái niệm cơ bản

### Biến và kiểu dữ liệu
```python
# Biến
name = "Nguyễn Văn Trung"
age = 25

# Kiểu dữ liệu
number = 10  # int
price = 19.99  # float
is_student = True  # boolean
items = ['apple', 'banana', 'orange']  # list
```

### Hàm
```python
def greet(name):
    return f"Hello, {name}!"

result = greet("Trung")
print(result)  # Output: Hello, Trung!
```

### Vòng lặp
```python
# For loop
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# While loop
count = 0
while count < 3:
    print(count)
    count += 1
```

## Thư viện quan trọng cho Data Analysis

- **Pandas**: Xử lý dữ liệu dạng bảng (DataFrame)
- **NumPy**: Tính toán số học và mảng
- **Matplotlib**: Vẽ biểu đồ
- **Seaborn**: Vẽ biểu đồ thống kê
- **Scikit-learn**: Machine learning

## Kết luận

Python là ngôn ngữ lý tưởng để bắt đầu hành trình Data Analysis của bạn. Với cú pháp đơn giản và những thư viện mạnh mẽ, bạn có thể xử lý dữ liệu, tạo biểu đồ, và thậm chí xây dựng mô hình machine learning. Hãy bắt đầu học Python ngay hôm nay!
